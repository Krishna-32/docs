---
title: 'Text Randomizer'
description: "A component that creates an engaging text animation effect on hover.
<br/> <br/>
When the text is hovered over, each character rapidly cycles through random characters before settling back to the original text.
The randomization creates a dynamic scrambling effect that resolves smoothly character by character."
---
  

## Parameters
The TextRandomizer component accepts the following props:

| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| children | React.ReactNode | - | The text content to be animated |

Returns an animated text component that scrambles characters on hover.

## Import

```jsx
import { TextRandomizer } from "dimension-js";
```

## Example

```jsx
<div className="w-full h-screen flex items-center justify-center bg-black text-white text-5xl">
   <TextRandomizer>DimensionJS</TextRandomizer>
</div>
```
