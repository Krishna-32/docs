---
title: 'Infinite Text Menu'
description: "A component that creates an infinite scrolling text menu with smooth animations.
<br/> <br/>
The menu displays a list of titles and subtitles that can be scrolled infinitely in either direction.
Items dynamically scale and rotate based on scroll velocity for an engaging visual effect.
Supports both mouse wheel scrolling and touch/drag interactions for intuitive navigation."
---
  

## Parameters

The InfiniteTextMenu component accepts the following props:

| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| subtitle | string[] | [] | Array of subtitle texts to display |
| title | string[] | [] | Array of title texts to display |
| bgImage | string | "" | URL of the background image |
| subtitleColor | string | "#999" | Color of the subtitle text |
| titleColor | string | "#fff" | Color of the title text |
| width | string | "100vw" | Width of the menu container |
| height | string | "100vh" | Height of the menu container |
| subtitleFontSize | string | "10px" | Font size for subtitles with responsive scaling |
| titleFontSize | string | "20px" | Font size for titles with responsive scaling |

Returns a div containing the infinite scrolling menu.

## Import

```jsx
import { InfiniteTextMenu } from "dimension-js";
```

## Example

```jsx
<InfiniteTextMenu 
  subtitle={["01", "02", "03"]}
  title={["First", "Second", "Third"]}
  bgImage="/background.jpg"
  subtitleColor="#999"
  titleColor="#fff"
  width="100vw"
  height="100vh"
  subtitleFontSize="10px"
  titleFontSize="20px"
/>
```
