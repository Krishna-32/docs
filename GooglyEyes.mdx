---
title: 'Googly Eyes'
description: "A component that renders a pair of googly eyes that track mouse movement
<br/> <br/>
The eyes consist of white scleras with black pupils that follow the mouse cursor.
The pupils move realistically within the eye bounds based on mouse position."
---

## Parameters
The GooglyEyes component accepts the following props:

| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| eyeSize | string | "150px" | The width and height of each eye |
| eyeColor | string | "white" | The background color of the eye sclera |
| pupilSize | string | "60px" | The width and height of each pupil |
| pupilColor | string | "black" | The color of the pupils |
| borderWidth | string | "2px" | The width of the eye border |
| borderColor | string | "black" | The color of the eye border |
| eyeGap | string | "2rem" | The spacing between the two eyes |

Returns a div containing two interactive googly eyes that follow mouse movement.

## Import

```jsx
import { GooglyEyes } from "dimension-js";
```

## Example

```jsx
<GooglyEyes />
```

## Customization

```jsx
<GooglyEyes 
  eyeSize="200px"
  eyeColor="lightblue"
  pupilSize="80px"
  pupilColor="darkblue"
  borderWidth="4px"
  borderColor="#000"
  eyeGap="3rem"
/>
```
