---
title: 'Animated Line'
description: "A component that renders an animated line that responds to mouse movement.
<br/> <br/>
The line is implemented as an SVG path that deforms based on the mouse position.
When the mouse moves over the line, it curves toward or away from the mouse.
The line smoothly animates back to its original straight position when the mouse leaves."
---


## Parameters

The AnimatedLine component accepts the following props:

| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| strokeColor | string | "#d5d4d3" | The color of the line stroke |

Returns an SVG element containing an animated path that responds to mouse movement.

## Import

```jsx
import { AnimatedLine } from "dimension-js";
```

## Example

```jsx
<div className="w-full h-screen flex items-center justify-center">
  <AnimatedLine />
</div>
```

## Customization

```jsx
<div className="w-full h-screen flex items-center justify-center">
  <AnimatedLine strokeColor="#f74f4d" />
</div>
```
